Table Ranking Logic

Overview

The league table is sorted using the following criteria, in order:

  1. Total Points (descending)
  2. Trophies (Wins) (descending)
  3. Net Win Percentage (descending)
  4. Total Winnings (descending)
  5. Alphabetical (A–Z by player name)

Tie‑Breaker Example 1

Players with 26 Total Points:

Rank  Name                Points  Trophies  Win %  Winnings
4     Paul Mellor          26      0         0.0%   £16.32
5     Jay Dempsey          26      0         0.0%   £ 0.00
6     Simon Peers          26      0         0.0%   £ 0.00

All three are tied on points, trophies and win %. We compare total winnings next:

  • Paul Mellor has £16.32 → ranked highest (#4).
  • Jay Dempsey and Simon Peers both have £0.00 → still tied.
  • Between those two, we use alphabetical order of full names:
    • Jay Dempsey ("J") comes before Simon Peers ("S").

Tie‑Breaker Example 2

Players with 20 Total Points:

11    Tony Doyle           20      0         0.0%   £ 0.00

They’re tied on points, trophies, win %, and winnings. Final tie‑breaker:

  • Compare names alphabetically: Jim Hall ("J") comes before Tony Doyle ("T").
